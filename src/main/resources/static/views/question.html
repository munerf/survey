Recomendações


Em cada par, as palavras apresentadas poderão pertencer à mesma língua ou a línguas diferentes. A língua de cada palavra estará assinalada pela respectiva bandeira. Lembre-se: as palavras portuguesas correspondem à ortografia anterior ao Acordo Ortográfico.

Numa leitura preliminar é possível que confunda algumas palavras, por exemplo que leia uma palavra inglesa com pronúncia alemã. Preste atenção para não cometer erros, não será permitido retroceder.

Irá encontrar duas opções ‘Desconheço’, uma para cada palavra. Deverá selecioná-la(s) sempre que desconhecer a(s) respetiva(s) palavra(s). Para fazer as suas avaliações terá de conhecer o significado das duas palavras. Por isso, se em cada par selecionar pelo menos uma opção ’Desconheço’, não lhe será permitido avaliar esse par. Não se esqueça: se desconhecer as duas palavras deverá carregar em ambos os botões ‘Desconheço’.
